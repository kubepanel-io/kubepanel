How Rate Limits Work

Rate Limit Format
  • 50 / 1h = 50 emails per hour
  • 100 / 30m = 100 emails per 30 minutes
  • 500 / 1d = 500 emails per day
Enforcement
  • Rate limits are enforced by rspamd in real-time
  • Counters are stored in Redis
  • Limits reset after the specified period
Priority
  • Custom user limits override global defaults
  • Removing a custom limit reverts to the default
  • Changes take effect immediately
Tips
  • Set lower limits for untrusted users
  • Increase limits for legitimate bulk senders
  • Monitor traffic to identify appropriate limits
