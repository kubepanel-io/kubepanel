How L3 Firewall Rules Work

L3 vs L7 Firewall
  • L3 (Network layer): Blocks at IP/port level, very efficient
  • L7 (Application layer): Blocks at HTTP level (WAF rules)
  • L3 rules apply to ALL traffic, not just HTTP
Source Matching
  • nets: Match traffic FROM these CIDRs
  • notNets: Match traffic NOT from these CIDRs
  • CIDR format: 10.0.0.0/8, 192.168.1.100/32
Actions
  • Deny: Block matching traffic at network layer
  • Allow: Explicitly allow matching traffic
  • Default: Allow all (if no rules match)
Examples
  • Block IP 1.2.3.4/32 on all ports
  • Block external SSH: Deny ports 22 from NOT 10.0.0.0/8
  • Block CIDR range 45.0.0.0/8
