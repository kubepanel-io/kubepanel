Preferred Servers

How to move your workload to a different server:

  1. Select your preferred server(s) and click Save Changes
  2. Stop the domain using the suspend button
  3. Wait until the domain is fully stopped (check Live Cluster Status - the web deployment should disappear)
  4. Start the domain again - it will now run on your preferred server

Why? The storage volume can only be attached to one server at a time. The workload must fully stop before it can be moved to a different server.

Select which servers this domain should preferably run on. If none selected, any available server will be used.
