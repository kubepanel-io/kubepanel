How SMTP Firewall Works

Blocking Types
  • Sender: Block specific email address (e.g., spammer@example.com)
  • Domain: Block all emails from a domain (e.g., @example.com)
  • IP: Block all emails from an IP address
Rate Limiting
  • Automatic limits per authenticated user
  • Per-domain limits prevent mass mailing
  • Exceeding limits results in temporary blocks
Powered by rspamd
  • Industry-standard spam detection
  • Real-time rate limit enforcement
  • Redis-backed persistent storage
Tips
  • Monitor traffic first, then block problematic senders
  • Use domain blocks carefully - they affect all users
  • Blocks take effect immediately
